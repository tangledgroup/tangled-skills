# Data Objects and Annotations

## Document

Holds annotations for an entire document. Created automatically when the Pipeline processes text.

**Properties:**
- `text` (str): Raw text of the document
- `sentences` (List[Sentence]): List of sentences
- `entities` / `ents` (List[Span]): Named entities in the document
- `num_tokens` (int): Total tokens
- `num_words` (int): Total words

**Methods:**
- `iter_words()` â†’ Iterator[Word]: All words in order
- `iter_tokens()` â†’ Iterator[Token]: All tokens in order
- `to_dict()` â†’ List[List[Dict]]: Serialize to nested dict list
- `to_serialized()` â†’ bytes: Pickle serialization
- `from_serialized(data)` â†’ Document: Class method to deserialize

## Sentence

Represents a single sentence as segmented by the TokenizeProcessor.

**Properties:**
- `doc` (Document): Back-pointer to parent document
- `text` (str): Raw text of the sentence
- `dependencies` (List[(Word, str, Word)]): Dependency triples of (head_word, relation, dependent_word)
- `tokens` (List[Token]): Tokens in this sentence
- `words` (List[Word]): Words in this sentence
- `entities` / `ents` (List[Span]): Named entities in this sentence
- `sentiment` (int): Sentiment score (0=negative, 1=neutral, 2=positive)
- `constituency` (ParseTree): Constituency parse tree
- `sent_id` (str): Sentence ID

**Methods:**
- `to_dict()` â†’ List[Dict]: Serialize sentence tokens
- `print_dependencies()`: Print syntactic dependencies
- `print_tokens()`: Print tokens
- `print_words()`: Print words
- `tokens_string()` â†’ str: Tokens as string
- `words_string()` â†’ str: Words as string

## Token

A surface-level token from the tokenizer. Wraps one or more Words (for multi-word tokens).

**Properties:**
- `id` (Tuple[int]): 1-based index. For MWTs, contains a range like `(3, 4)`.
- `text` (str): Text of this token
- `misc` (str): Miscellaneous annotations
- `words` (List[Word]): Underlying syntactic words
- `start_char` (int): Start character offset in document
- `end_char` (int): End character offset in document
- `ner` (str): BIOES NER tag, e.g., `'B-ORG'`
- `spaces_after` (str): Space(s) following the token
- `spaces_before` (str): Space(s) before the token

**Methods:**
- `to_dict()` â†’ List[Dict]: Serialize underlying words
- `pretty_print()` â†’ str: Print token with expanded words

## Word

A syntactic word with all word-level annotations. Generated by MWT expansion from Tokens.

**Properties:**
- `id` (int): 1-based index in sentence (0 reserved for artificial root)
- `text` (str): Text of the word
- `lemma` (str): Lemma form
- `upos` / `pos` (str): Universal POS tag, e.g., `'NOUN'`
- `xpos` (str): Treebank-specific POS tag, e.g., `'NNP'`
- `feats` (str): Morphological features, e.g., `'Gender=Fem|Person=3'`
- `head` (int): ID of syntactic head (0 = root)
- `deprel` (str): Dependency relation, e.g., `'nmod'`
- `deps` (str): Enhanced dependencies from CoNLL-U
- `misc` (str): Miscellaneous annotations
- `parent` (Token): Back-pointer to parent Token

**Methods:**
- `to_dict()` â†’ Dict: Serialize word information
- `pretty_print()` â†’ str: Print word with all annotations

## Span

Represents a contiguous span of text, used for named entities.

**Properties:**
- `doc` (Document): Parent document
- `text` (str): Text of the span
- `tokens` (List[Token]): Tokens in this span
- `words` (List[Word]): Words in this span
- `type` (str): Entity type, e.g., `'PERSON'`
- `start_char` (int): Start character offset
- `end_char` (int): End character offset

**Methods:**
- `to_dict()` â†’ Dict: Serialize span
- `pretty_print()` â†’ str: Print span information

## ParseTree

Nested tree structure from the constituency parser.

**Properties:**
- `label` (str): Bracket type for inner nodes, POS tag for preterminals, word text for leaves
- `children` (List[ParseTree]): Child nodes

Example traversal:
```python
tree = doc.sentences[0].constituency
print(tree.label)           # 'ROOT'
print(tree.children[0])     # (S (NP ...) (VP ...))
```

## Adding Custom Properties

Extend any Stanza data object with custom annotations:

```python
Document.add_property('char_count',
    default=0,
    getter=lambda self: len(self.text),
    setter=None)

# Now accessible on all Document instances
doc.char_count  # returns length of doc.text
```

Properties are read-only by default unless a `setter` is provided. The underlying variable is named `_{property_name}`.

## Data Conversion

Convert between Stanza Documents, Python objects, and CoNLL-U format:

```python
# Document to Python dict
dicts = doc.to_dict()  # List[List[Dict]]

# Python dict to Document
from stanza.models.common.doc import Document
doc = Document(dicts)

# Read/write CoNLL-U
from stanza.utils.conconll import read_conllu, write_conllu
```
