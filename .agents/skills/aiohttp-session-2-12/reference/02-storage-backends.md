# Storage Backends Reference

This document covers all 5 storage backends provided by aiohttp-session 2.12.1, including configuration options, examples, and backend-specific features.

## AbstractStorage Base Class

All storage backends inherit from `AbstractStorage` which provides common functionality:

**Common Parameters:**
```python
class AbstractStorage:
    def __init__(
        self,
        *,
        cookie_name: str = "AIOHTTP_SESSION",  # Cookie name for session ID
        domain: Optional[str] = None,           # Cookie domain
        max_age: Optional[int] = None,          # Cookie/session lifetime in seconds
        path: str = "/",                        # Cookie path
        secure: Optional[bool] = None,          # Secure flag (HTTPS only)
        httponly: bool = True,                  # HttpOnly flag (default True)
        samesite: Optional[str] = None,         # SameSite attribute (Lax/Strict/None)
        encoder: Callable = json.dumps,         # Custom serializer
        decoder: Callable = json.loads,         # Custom deserializer
    )
```

**Common Properties:**
- `cookie_name` - Name of the cookie used for session storage
- `max_age` - Maximum age in seconds (None = browser session cookie)
- `cookie_params` - Dict of all cookie parameters

**Abstract Methods (implemented by subclasses):**
- `async load_session(request)` - Load session from storage
- `async save_session(request, response, session)` - Save session to storage

**Helper Methods:**
- `load_cookie(request)` - Extract cookie value from request
- `save_cookie(response, cookie_data, max_age)` - Set cookie in response
- `new_session()` - Create a new empty Session object

## SimpleCookieStorage

**⚠️ WARNING: For testing only! Never use in production.**

Stores session data as unencrypted JSON directly in the cookie. Anyone can read and modify session data.

### Installation

No additional dependencies required:

```bash
pip install aiohttp-session
```

### Configuration

```python
from aiohttp_session import setup
from aiohttp_session import SimpleCookieStorage

app = web.Application()
setup(app, SimpleCookieStorage())
```

**Parameters:** Same as `AbstractStorage` (cookie_name, domain, max_age, path, secure, httponly, samesite, encoder, decoder)

### Example

```python
from aiohttp import web
from aiohttp_session import setup, get_session, SimpleCookieStorage

async def handler(request):
    session = await get_session(request)
    session['counter'] = session.get('counter', 0) + 1
    return web.Response(text=f'Count: {session["counter"]}')

app = web.Application()
setup(app, SimpleCookieStorage())  # ⚠️ INSECURE - testing only!
app.router.add_get('/', handler)
```

### Security Considerations

**DO NOT USE in production because:**
- Session data is visible to anyone (base64-encoded JSON in cookie)
- No integrity protection (can be modified)
- No encryption (sensitive data exposed)
- Vulnerable to session hijacking

**Use only for:**
- Local development and testing
- Prototyping without security requirements
- Internal tools with no sensitive data

## EncryptedCookieStorage

**RECOMMENDED for most applications.**

Stores session data in cookies encrypted with Fernet (AES-128-CBC + HMAC). Self-contained sessions with no server-side storage required.

### Installation

```bash
pip install 'aiohttp-session[secure]'
# Installs: aiohttp-session + cryptography
```

### Key Generation

Generate a proper 32-byte Fernet key:

```python
from cryptography.fernet import Fernet

# Generate new key (do this once, save securely)
key = Fernet.generate_key()
print(key.decode())  # Save this! e.g., "gJ5...=="

# Or use existing 32-byte key
key = b'Thirty  two  length  bytes  key.'  # Exactly 32 bytes
```

### Configuration

**With Fernet instance:**

```python
from cryptography.fernet import Fernet
from aiohttp_session import setup
from aiohttp_session.cookie_storage import EncryptedCookieStorage

fernet_key = Fernet.generate_key()
fernet = Fernet(fernet_key)

app = web.Application()
setup(app, EncryptedCookieStorage(fernet))
```

**With bytes key (auto-creates Fernet):**

```python
secret_key = b'32-byte-secret-key-here-1234567890'  # Exactly 32 bytes

app = web.Application()
setup(app, EncryptedCookieStorage(secret_key))
```

**With base64-encoded string:**

```python
# Key generated by Fernet.generate_key() returns base64 string
secret_key = "gJ5VxKjH8mN3pL2qR9sT1uV4wX6yZ0aB3cD5eF7gH9i="

app = web.Application()
setup(app, EncryptedCookieStorage(secret_key))
```

### Full Example

```python
import time
from cryptography.fernet import Fernet
from aiohttp import web
from aiohttp_session import setup, get_session
from aiohttp_session.cookie_storage import EncryptedCookieStorage

async def handler(request):
    session = await get_session(request)
    
    # Track visits
    if 'visits' not in session:
        session['visits'] = 1
        session['first_visit'] = time.time()
    else:
        session['visits'] += 1
    
    return web.Response(text=f'''
        Visit count: {session["visits"]}
        First visit: {time.ctime(session["first_visit"])}
    ''')

def make_app():
    app = web.Application()
    
    # Production: Load key from environment variable
    fernet_key = Fernet.generate_key()  # Or os.environ['SESSION_KEY']
    setup(app, EncryptedCookieStorage(fernet_key))
    
    app.router.add_get('/', handler)
    return app

web.run_app(make_app())
```

### Configuration Options

```python
from aiohttp_session.cookie_storage import EncryptedCookieStorage

storage = EncryptedCookieStorage(
    secret_key=b'32-byte-secret-key-here-1234567890',
    cookie_name="AIOHTTP_SESSION",  # Cookie name
    max_age=1800,                    # 30 minutes (seconds)
    path="/",                        # Cookie path
    domain=None,                     # Domain (None = current domain)
    secure=True,                     # HTTPS only
    httponly=True,                   # No JavaScript access
    samesite="Lax"                   # CSRF protection (Lax/Strict/None)
)
```

### Features

✅ **Pros:**
- No server-side storage required (stateless)
- Works great for horizontal scaling
- Encrypted and authenticated data
- Simple deployment

❌ **Cons:**
- Cookie size limited (~4KB total, ~3.5KB encrypted data)
- Can't invalidate single session (must rotate keys)
- Key rotation requires migration strategy

### TTL Support

EncryptedCookieStorage respects `max_age` for both cookie expiry and token TTL:

```python
storage = EncryptedCookieStorage(
    secret_key=fernet_key,
    max_age=3600  # Session expires after 1 hour
)

# Also supports per-session max_age
async def handler(request):
    session = await get_session(request)
    session.max_age = 900  # This session expires in 15 minutes
```

If token TTL expires, a new session is automatically created.

## NaClCookieStorage

Alternative encrypted cookie storage using libsodium (NaCl) SecretBox encryption.

### Installation

```bash
pip install pynacl
pip install aiohttp-session
```

### Configuration

```python
from aiohttp_session import setup
from aiohttp_session.nacl_storage import NaClCookieStorage

# Requires exactly 32-byte key
secret_key = b'32-byte-secret-key-here-1234567890'

app = web.Application()
setup(app, NaClCookieStorage(secret_key))
```

### Full Example

```python
from aiohttp import web
from aiohttp_session import setup, get_session
from aiohttp_session.nacl_storage import NaClCookieStorage

async def handler(request):
    session = await get_session(request)
    session['data'] = 'encrypted with NaCl'
    return web.Response(text='OK')

def make_app():
    app = web.Application()
    
    # Generate 32-byte key (use secrets.token_bytes(32))
    secret_key = b'32-byte-secret-key-here-1234567890'
    
    setup(app, NaClCookieStorage(secret_key))
    app.router.add_get('/', handler)
    return app

web.run_app(make_app())
```

### Comparison with EncryptedCookieStorage

| Feature | Fernet (EncryptedCookieStorage) | NaCl (NaClCookieStorage) |
|---------|--------------------------------|--------------------------|
| Library | cryptography | pynacl (libsodium) |
| Encryption | AES-128-CBC + HMAC | XSalsa20 + Poly1305 |
| Key size | 32 bytes | 32 bytes |
| TTL support | ✅ Yes | ❌ No |
| Corrupted cookie handling | ✅ Auto-create new session | ✅ Auto-create new session |

**Recommendation:** Use EncryptedCookieStorage (Fernet) for better ecosystem support and TTL features.

## RedisStorage

Stores session data in Redis server, keeping only a UUID key in the cookie. Ideal for large sessions or distributed applications.

### Installation

```bash
pip install 'aiohttp-session[aioredis]'
# Installs: aiohttp-session + redis (>=4.3)
```

**Note:** aiohttp-session 2.12.0+ migrated from `aioredis` to `redis.asyncio`. Requires redis-py >= 4.3.

### Configuration

```python
import asyncio
from redis.asyncio import from_url
from aiohttp import web
from aiohttp_session import setup, get_session
from aiohttp_session.redis_storage import RedisStorage

async def make_app():
    app = web.Application()
    
    # Connect to Redis
    redis = await from_url("redis://localhost:6379", decode_responses=False)
    
    setup(app, RedisStorage(redis))
    
    async def handler(request):
        session = await get_session(request)
        session['user_data'] = {'name': 'John', 'role': 'admin'}
        return web.Response(text='Session saved to Redis')
    
    app.router.add_get('/', handler)
    return app

asyncio.run(web.run_app(make_app()))
```

### Redis Connection Options

**Basic connection:**

```python
redis = await from_url("redis://localhost:6379")
storage = RedisStorage(redis)
```

**With authentication:**

```python
redis = await from_url("redis://:password@localhost:6379")
storage = RedisStorage(redis)
```

**With database selection:**

```python
redis = await from_url("redis://localhost:6379/2")  # DB 2
storage = RedisStorage(redis)
```

**With connection pool:**

```python
redis = await from_url(
    "redis://localhost:6379",
    max_connections=10,
    health_check_interval=30
)
storage = RedisStorage(redis)
```

### Configuration Options

```python
from aiohttp_session.redis_storage import RedisStorage

storage = RedisStorage(
    redis_pool=redis,
    cookie_name="AIOHTTP_SESSION",     # Cookie name
    max_age=1800,                       # Session lifetime (seconds)
    path="/",
    domain=None,
    secure=True,
    httponly=True,
    samesite="Lax",
    key_factory=lambda: uuid.uuid4().hex  # Custom key generator
)
```

**key_factory:** Custom function to generate session keys (default: UUID hex)

```python
import uuid

# Custom key factory
def custom_key():
    return f"sess_{uuid.uuid4().hex}"

storage = RedisStorage(redis, key_factory=custom_key)
```

### Redis Key Format

Session data is stored in Redis with key format: `{cookie_name}_{session_id}`

Example:
- Cookie in browser: `AIOHTTP_SESSION=e33b57c7ec6e425eb626610f811ab6ae`
- Redis key: `AIOHTTP_SESSION_e33b57c7ec6e425eb626610f811ab6ae`
- Redis value: `{"created": 1234567890, "session": {"user_id": 123, "username": "john"}}`

### Session Expiration

Redis keys automatically expire based on `max_age`:

```python
storage = RedisStorage(redis, max_age=3600)  # Keys expire after 1 hour

# Per-session expiration
async def handler(request):
    session = await get_session(request)
    session.max_age = 900  # This session expires in 15 minutes
    session['data'] = 'short-lived'
```

### Features

✅ **Pros:**
- No cookie size limitations
- Can invalidate individual sessions
- Centralized session storage
- Works across multiple servers
- Session persistence across browser restarts

❌ **Cons:**
- Requires Redis server
- Additional network latency
- Single point of failure (use Redis Cluster for HA)

### Cleanup Stale Sessions

Redis automatically expires keys, but you can manually cleanup:

```python
# Delete specific session
await redis.delete(f"AIOHTTP_SESSION_{session_id}")

# Pattern-based cleanup (delete all sessions older than X days)
# Use Redis SCAN with timestamp checking in created field
```

Similar to Redis: `{cookie_name}_{session_id}`

Example:
- Cookie: `AIOHTTP_SESSION=abc123def456`
- Memcached key: `AIOHTTP_SESSION_abc123def456`

### Features

✅ **Pros:**

- Fast in-memory storage
- Can invalidate individual sessions
- Works across multiple servers

❌ **Cons:**
- Requires Memcached server
- No persistence (data lost on restart)
- Less feature-rich than Redis

## Custom Storage Backend

Create a custom storage by implementing `AbstractStorage`. This section includes examples for PostgreSQL and other databases.

### PostgreSQL Storage Example

Complete implementation using aiopg for async PostgreSQL connections:

```python
import json
import uuid
from typing import Any, Callable, Optional

import psycopg2.extras
from aiohttp import web
from aiopg import Pool

from aiohttp_session import AbstractStorage, Session


class PgStorage(AbstractStorage):
    """PostgreSQL session storage.
    
    Requires:
    - aiopg: pip install aiopg
    - psycopg2-binary: pip install psycopg2-binary
    
    Database schema:
    ```sql
    CREATE SCHEMA web;
    CREATE TABLE web.sessions (
        uuid UUID PRIMARY KEY,
        session JSONB NOT NULL,
        created TIMESTAMP NOT NULL,
        expire TIMESTAMP NOT NULL
    );
    CREATE INDEX idx_sessions_expire ON web.sessions (expire);
    """

    def __init__(
        self,
        pg_pool: Pool,
        *,
        cookie_name: str = "AIOHTTP_SESSION",
        domain: Optional[str] = None,
        max_age: Optional[int] = None,
        path: str = "/",
        secure: Optional[bool] = None,
        httponly: bool = True,
        key_factory: Callable[[], str] = lambda: uuid.uuid4().hex,
        encoder: Callable[[object], str] = psycopg2.extras.Json,
        decoder: Callable[[str], Any] = json.loads,
    ):
        super().__init__(
            cookie_name=cookie_name,
            domain=domain,
            max_age=max_age,
            path=path,
            secure=secure,
            httponly=httponly,
            encoder=encoder,
            decoder=decoder,
        )
        self._pg = pg_pool
        self._key_factory = key_factory

    async def load_session(self, request: web.Request) -> Session:
        cookie = self.load_cookie(request)
        if cookie is None:
            return Session(None, data={}, new=True, max_age=self.max_age)
        else:
            async with self._pg.acquire() as conn:
                key = uuid.UUID(cookie)
                async with conn.cursor(
                    cursor_factory=psycopg2.extras.DictCursor
                ) as cur:
                    await cur.execute(
                        "SELECT session, extract(epoch from created) "
                        + "FROM web.sessions WHERE uuid = %s",
                        (key,),
                    )
                    data = await cur.fetchone()

                    if not data:
                        return Session(None, data={}, new=True, max_age=self.max_age)

            return Session(key, data=data, new=False, max_age=self.max_age)

    async def save_session(
        self, request: web.Request, response: web.StreamResponse, session: Session
    ) -> None:
        key = session.identity
        if key is None:
            key = self._key_factory()
            self.save_cookie(response, key, max_age=session.max_age)
        else:
            if session.empty:
                self.save_cookie(response, "", max_age=session.max_age)
            else:
                key = str(key)
                self.save_cookie(response, key, max_age=session.max_age)

        data = self._get_session_data(session)
        if not data:
            return

        data_encoded = self._encoder(data["session"])
        expire = data["created"] + (session.max_age or 0)
        async with self._pg.acquire() as conn:
            async with conn.cursor() as cur:
                await cur.execute(
                    "INSERT INTO web.sessions (uuid,session,created,expire)"
                    + " VALUES (%s, %s, to_timestamp(%s),to_timestamp(%s))"
                    + " ON CONFLICT (uuid)"
                    + " DO UPDATE"
                    + " SET (session,expire)=(EXCLUDED.session, EXCLUDED.expire)",
                    [key, data_encoded, data["created"], expire],
                )


# Usage example
import asyncio
from aiohttp import web
from aiohttp_session import setup
from aiopg import create_pool


async def handler(request: web.Request):
    from aiohttp_session import get_session
    session = await get_session(request)
    session["data"] = "stored in PostgreSQL"
    return web.Response(text="OK")


async def make_app():
    app = web.Application()
    
    # Create PostgreSQL connection pool
    pool = await create_pool(
        host='localhost',
        port=5432,
        user='postgres',
        password='password',
        database='myapp'
    )
    
    # Setup session storage
    storage = PgStorage(pool, max_age=3600)
    setup(app, storage)
    
    app.router.add_get("/", handler)
    
    # Cleanup pool on shutdown
    app.on_cleanup.append(lambda app: pool.close())
    
    return app


if __name__ == "__main__":
    loop = asyncio.get_event_loop()
    app = loop.run_until_complete(make_app())
    web.run_app(app)
```

##### Generic Database Storage Template

Use this template as a starting point for other databases (MySQL, SQLite, MongoDB, etc.):

```python
from aiohttp_session import AbstractStorage, Session
from aiohttp import web
import json
import uuid


class DatabaseStorage(AbstractStorage):
    """Generic database-backed session storage template.
    
    Adapt the SQL queries and connection handling for your database.
    """
    
    def __init__(self, db_connection, key_factory=None, **kwargs):
        super().__init__(**kwargs)
        self.db = db_connection
        self._key_factory = key_factory or (lambda: uuid.uuid4().hex)
    
    async def load_session(self, request: web.Request) -> Session:
        cookie_value = self.load_cookie(request)
        
        if not cookie_value:
            return Session(None, data={}, new=True, max_age=self.max_age)
        
        # Query database for session data
        # Adapt this query for your database
        row = await self.db.fetch(
            "SELECT data, created FROM sessions WHERE id = $1", 
            cookie_value
        )
        
        if not row:
            return Session(None, data={}, new=True, max_age=self.max_age)
        
        # Parse data and create session
        data = {
            "session": json.loads(row['data']),
            "created": row['created'].timestamp()
        }
        return Session(cookie_value, data=data, new=False, max_age=self.max_age)
    
    async def save_session(
        self, request: web.Request, 
        response: web.StreamResponse, 
        session: Session
    ) -> None:
        key = session.identity
        if key is None:
            key = self._key_factory()
            self.save_cookie(response, key, max_age=session.max_age)
        else:
            if session.empty:
                # Delete session from database
                await self.db.execute(
                    "DELETE FROM sessions WHERE id = $1", key
                )
                self.save_cookie(response, "", max_age=session.max_age)
                return
            else:
                self.save_cookie(response, key, max_age=session.max_age)

        # Prepare session data
        data = self._get_session_data(session)
        if not data:
            return

        # Upsert session data (adapt for your database)
        data_encoded = self._encoder(data["session"])
        expire = data["created"] + (session.max_age or 0)
        
        await self.db.execute(
            """INSERT INTO sessions (id, data, created, expire)
               VALUES ($1, $2, $3, $4)
               ON CONFLICT (id) DO UPDATE 
               SET (data, expire) = ($2, $4)""",
            key, data_encoded, data["created"], expire
        )
```

### Custom Encoder/Decoder

For non-JSON data or custom serialization needs:

**Pickle for Complex Objects:**
```python
import base64
import pickle

def pickle_dumps(obj):
    return base64.b64encode(pickle.dumps(obj)).decode('ascii')

def pickle_loads(data):
    return pickle.loads(base64.b64decode(data.encode('ascii')))

storage = EncryptedCookieStorage(
    secret_key=fernet_key,
    encoder=pickle_dumps,
    decoder=pickle_loads
)

# Now you can store custom objects in session
session['custom_obj'] = MyCustomClass()
```

**⚠️ Security Warning:** Pickle is vulnerable to arbitrary code execution. Only use with trusted data.

**MsgPack for Better Performance:**
```python
import msgpack

def msgpack_dumps(obj):
    return msgpack.packb(obj, use_bin_type=True)

def msgpack_loads(data):
    return msgpack.unpackb(data, raw=False, strict_map_key=False)

storage = RedisStorage(
    redis,
    encoder=msgpack_dumps,
    decoder=msgpack_loads
)
```

**Custom Type Serialization:**
```python
from datetime import datetime
import json

def custom_encoder(obj):
    """Handle datetime and other custom types."""
    if isinstance(obj, datetime):
        return {"__type__": "datetime", "value": obj.isoformat()}
    elif hasattr(obj, "to_dict"):
        return {"__type__": "object", "value": obj.to_dict()}
    raise TypeError(f"Object of type {type(obj)} is not JSON serializable")

def custom_decoder(dct):
    """Restore custom types from JSON."""
    if "__type__" in dct:
        if dct["__type__"] == "datetime":
            return datetime.fromisoformat(dct["value"])
        elif dct["__type__"] == "object":
            return CustomObject.from_dict(dct["value"])
    return dct

storage = EncryptedCookieStorage(
    secret_key=fernet_key,
    encoder=lambda obj: json.dumps(obj, default=custom_encoder),
    decoder=lambda s: json.loads(s, object_hook=custom_decoder)
)

# Now you can store datetime objects
session['last_login'] = datetime.now()
```

**Compression for Large Sessions:**
```python
import json
import zlib
import base64

def compressed_dumps(obj):
    """Compress JSON data to reduce cookie/storage size."""
    json_str = json.dumps(obj)
    compressed = zlib.compress(json_str.encode('utf-8'))
    return base64.b64encode(compressed).decode('ascii')

def compressed_loads(data):
    """Decompress and parse JSON data."""
    decompressed = zlib.decompress(base64.b64decode(data.encode('ascii')))
    return json.loads(decompressed.decode('utf-8'))

storage = EncryptedCookieStorage(
    secret_key=fernet_key,
    encoder=compressed_dumps,
    decoder=compressed_loads
)
```
