# ASN.1 Encoding and Decoding

The `cryptography.hazmat.asn1` module provides declarative ASN.1 type definitions with encoding and decoding support.

**Note**: These APIs are considered **unstable** and not yet subject to the backwards compatibility policy (since 47.0.0).

**Warning**: This is a "Hazardous Materials" module. Only use if you understand ASN.1 structure and security implications.

## Tutorial

### Type Mapping

ASN.1 types map to Python constructs:

| ASN.1 Type | Python Class |
|---|---|
| INTEGER | `asn1.Integer` |
| OCTET STRING | `asn1.OctetString` |
| UTF8String | `asn1.UTF8String` |
| SEQUENCE | `asn1.Sequence` |
| SET | `asn1.Set` |
| CHOICE | `asn1.Choice` |
| BOOLEAN | `asn1.Boolean` |
| NULL | `asn1.Null` |
| OBJECT IDENTIFIER | `asn1.ObjectIdentifier` |

### Defining a SEQUENCE

```python
from cryptography.hazmat import asn1

class MySequence(asn1.Sequence):
    class ComponentTypes:
        name = asn1.UTF8String()
        value = asn1.Integer()
```

### Encoding and Decoding

```python
# Encode
instance = MySequence(name="example", value=42)
der_bytes = asn1.encode(instance)

# Decode
decoded = asn1.decode(MySequence, der_bytes)
print(decoded.name)   # "example"
print(decoded.value)  # 42
```

### Nested Sequences

```python
class Inner(asn1.Sequence):
    class ComponentTypes:
        code = asn1.Integer()

class Outer(asn1.Sequence):
    class ComponentTypes:
        label = asn1.UTF8String()
        inner = Inner()
```

### OPTIONAL Fields

Use `asn1.optional()` to mark fields as optional:

```python
class OptionalSeq(asn1.Sequence):
    class ComponentTypes:
        required = asn1.Integer()
        optional_field = asn1.optional(asn1.UTF8String())
```

### DEFAULT Values

```python
class DefaultSeq(asn1.Sequence):
    class ComponentTypes:
        name = asn1.UTF8String()
        version = asn1.default(asn1.Integer(), 1)
```

### CHOICE Fields

```python
class ChoiceType(asn1.Choice):
    class ComponentTypes:
        text = asn1.UTF8String()
        number = asn1.Integer()
```

### EXPLICIT and IMPLICIT Tagging

```python
class TaggedSeq(asn1.Sequence):
    class ComponentTypes:
        explicit_field = asn1.explicit(asn1.Integer(), tag=0)
        implicit_field = asn1.implicit(asn1.UTF8String(), tag=1)
```

### SEQUENCE OF and SET OF

```python
class SeqOf(asn1.SequenceOf):
    component_type = asn1.Integer()

class SetOf(asn1.SetOf):
    component_type = asn1.OctetString()
```

### Size Constraints

```python
class ConstrainedSeq(asn1.Sequence):
    class ComponentTypes:
        short_name = asn1.size(asn1.UTF8String(), min=1, max=64)
```

## ASN.1 Reference

### Serialization

- `asn1.encode(value)` — Encode an ASN.1 value to DER bytes
- `asn1.decode(type_, data)` — Decode DER bytes into an ASN.1 value of the given type

### Annotations

- `@asn1.tag(tag_number, explicit=True/False)` — Apply custom tagging to a class
- `@asn1.constraint(...)` — Apply constraints at the class level

## Use Cases

- Building custom X.509 extensions
- Working with proprietary ASN.1-based protocols
- Implementing cryptographic protocol message formats
- Encoding/decoding certificate structures not covered by the x509 module

## Best Practices

- Treat these APIs as experimental — they may change without backwards compatibility guarantees
- Validate decoded values before use in security-critical contexts
- Prefer the built-in `x509` module for standard certificate operations
- Use DER encoding (not BER) for interoperability
