## Usage Examples by Language Family

### Germanic Languages

```python
from lingua import Language, LanguageDetectorBuilder

# Build detector for Germanic languages only
germanic = [
    Language.ENGLISH,
    Language.GERMAN,
    Language.DUTCH,
    Language.SWEDISH,
    Language.NORWEGIAN_BOKMAL,
    Language.NORWEGIAN_NYNORSK,
    Language.DANISH,
    Language.ICELANDIC,
    Language.AFRIKAANS,
    Language.FINNISH  # Uralic but often grouped
]

detector = LanguageDetectorBuilder.from_languages(*germanic).build()

texts = [
    "Hello world",        # English
    "Hallo Welt",         # German
    "Hallo wereld",       # Dutch
    "Hej världen",        # Swedish
    "Hei verden",         # Norwegian Bokmål
]

for text in texts:
    lang = detector.detect_language_of(text)
    print(f"{text}: {lang.name if lang else 'Unknown'}")
```

# European languages

### By ISO Code

```python
from lingua import LanguageDetectorBuilder, IsoCode639_1, IsoCode639_3

# Using ISO 639-1 codes
detector = LanguageDetectorBuilder.from_iso_codes_639_1(
    IsoCode639_1.EN,
    IsoCode639_1.FR,
    IsoCode639_1.DE,
    IsoCode639_1.ES,
    IsoCode639_1.IT
).build()

# Using ISO 639-3 codes
detector = LanguageDetectorBuilder.from_iso_codes_639_3(
    IsoCode639_3.ENG,
    IsoCode639_3.FRA,
    IsoCode639_3.DEU,
    IsoCode639_3.ESP,
    IsoCode639_3.ITA
).build()
```

### Excluding Languages

```python
from lingua import Language, LanguageDetectorBuilder

# All languages except Latin and Esperanto
detector = LanguageDetectorBuilder.from_all_languages_without(
    Language.LATIN,
    Language.ESPERANTO
).build()

# All spoken languages (automatically excludes Latin)
detector = LanguageDetectorBuilder.from_all_spoken_languages().build()
```

## Similar Languages and Detection Challenges

Some language pairs are particularly challenging due to similarity:

| Language Pair | Challenge Level | Notes |
|--------------|-----------------|-------|
| Dutch - Afrikaans | High | Very similar vocabulary and grammar |
| Spanish - Portuguese | Medium-High | Similar Romance languages |
| Serbian - Croatian - Bosnian | High | Mutually intelligible variants |
| Norwegian Bokmål - Danish | Medium | Historical relationship |
| Czech - Slovak | Medium | West Slavic similarity |
| Swedish - Norwegian | Medium | North Germanic similarity |
| Bulgarian - Macedonian | High | Both South Slavic |

**Recommendation**: For similar languages, consider using minimum relative distance or confidence thresholds:

```python
from lingua import Language, LanguageDetectorBuilder

detector = LanguageDetectorBuilder.from_languages(
    Language.DUTCH, Language.AFRIKAANS
).with_minimum_relative_distance(0.5).build()

text = "Goedemore"  # Could be either
result = detector.detect_language_of(text)

if result is None:
    print("Could not reliably distinguish between Dutch and Afrikaans")
else:
    print(f"Detected: {result.name}")
```

## Complete Language Iteration

```python
from lingua import Language

# Iterate through all supported languages
print("All 75 supported languages:")
for language in sorted(Language.all()):
    iso_1 = language.iso_code_639_1.name if language.iso_code_639_1 else "N/A"
    iso_3 = language.iso_code_639_3.name if language.iso_code_639_3 else "N/A"
    print(f"{language.name:20} ISO-1: {iso_1:3} ISO-3: {iso_3}")

# Get count
print(f"\nTotal languages: {len(Language.all())}")  # 75
```

## Language from String Conversion

```python
from lingua import Language

# Case-insensitive conversion
lang1 = Language.from_str("english")   # Language.ENGLISH
lang2 = Language.from_str("ENGLISH")   # Language.ENGLISH
lang3 = Language.from_str("EngLish")   # Language.ENGLISH

# Works with any supported language name
lang4 = Language.from_str("german")    # Language.GERMAN
lang5 = Language.from_str("japanese")  # Language.JAPANESE

# Error handling - returns None for invalid names
try:
    lang = Language.from_str("invalid_language_name")
    print(lang)  # Behavior depends on implementation
except Exception as e:
    print(f"Error: {e}")
```
