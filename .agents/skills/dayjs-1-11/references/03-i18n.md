# Day.js Internationalization (i18n)

Day.js supports 80+ locales out of the box. Locales control date formats, weekday/month names, and relative time strings.

## Loading Locales

### Import and Set Global Locale

```javascript
import dayjs from 'dayjs'
import 'dayjs/locale/es' // Load Spanish locale

// Set globally
dayjs.locale('es')

// All instances now use Spanish
dayjs().format('LLLL') // 'martes, 15 de enero de 2024 14:30'
```

### Per-Instance Locale

```javascript
import dayjs from 'dayjs'
import 'dayjs/locale/es'
import 'dayjs/locale/fr'

// English (default)
dayjs().format('dddd') // 'Monday'

// Spanish instance
dayjs().locale('es').format('dddd') // 'lunes'

// French instance
dayjs().locale('fr').format('dddd') // 'lundi'

// Locale is preserved through operations
const spanishDate = dayjs().locale('es')
spanishDate.add(1, 'day').format('dddd') // 'martes' (still Spanish)
```

### Get Current Locale

```javascript
// Global locale
dayjs.locale() // 'en'

// Instance locale
dayjs().locale('es').locale() // 'es'
```

## Locale Configuration

### Creating Custom Locales

```javascript
import dayjs from 'dayjs'

dayjs.locale('mylocale', {
  // Weekday names
  weekdays: [
    'Sunday', 'Monday', 'Tuesday', 'Wednesday', 
    'Thursday', 'Friday', 'Saturday'
  ],
  weekdaysShort: ['Sun', 'Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat'],
  weekdaysMin: ['Su', 'Mo', 'Tu', 'We', 'Th', 'Fr', 'Sa'],
  
  // Month names
  months: [
    'January', 'February', 'March', 'April', 'May', 'June',
    'July', 'August', 'September', 'October', 'November', 'December'
  ],
  monthsShort: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun', 
                 'Jul', 'Aug', 'Sep', 'Oct', 'Nov', 'Dec'],
  
  // First day of week (0 = Sunday, 1 = Monday)
  weekStart: 1,
  
  // Relative time strings
  relativeTime: {
    future: 'in %s',
    past: '%s ago',
    s: 'a few seconds',
    m: 'a minute',
    mm: '%d minutes',
    h: 'an hour',
    hh: '%d hours',
    d: 'a day',
    dd: '%d days',
    M: 'a month',
    MM: '%d months',
    y: 'a year',
    yy: '%d years'
  },
  
  // Meridiem
  meridiem: function (hour, minute, isLower) {
    if (hour < 12) {
      return isLower ? 'am' : 'AM'
    } else {
      return isLower ? 'pm' : 'PM'
    }
  }
})

// Use custom locale
dayjs.locale('mylocale')
dayjs().format('LLLL') // Uses custom names
```

### Modifying Existing Locales

Requires `updateLocale` plugin:

```javascript
import dayjs from 'dayjs'
import updateLocale from 'dayjs/plugin/updateLocale'
dayjs.extend(updateLocale)

// Modify English locale
dayjs.updateLocale('en', {
  weekdays: ['Sun', 'Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat'],
  relativeTime: {
    future: 'in %s',
    past: '%s ago',
    s: 'seconds',
    m: 'a minute',
    mm: '%d minutes',
    h: 'an hour',
    hh: '%d hours',
    d: 'a day',
    dd: '%d days',
    M: 'a month',
    MM: '%d months',
    y: 'a year',
    yy: '%d years'
  }
})
```

### Locale Data Access

Requires `localeData` plugin:

```javascript
import dayjs from 'dayjs'
import localeData from 'dayjs/plugin/localeData'
dayjs.extend(localeData)

const data = dayjs().localeData()

// Get locale arrays
data.weekdays()     // ['Sunday', 'Monday', ...]
data.weekdaysShort() // ['Sun', 'Mon', ...]
data.weekdaysMin()   // ['Su', 'Mo', ...]

data.months()       // ['January', 'February', ...]
data.monthsShort()  // ['Jan', 'Feb', ...]

// Get first day of week
data.firstDayOfWeek() // 0 or 1

// Set first day of week
data.firstDayOfWeek(1) // Monday
```

## Locale-Specific Formatting

### Date Formats by Locale

Different locales format dates differently:

```javascript
import 'dayjs/locale/es'
import 'dayjs/locale/de'
import 'dayjs/locale/ja'
import localizedFormat from 'dayjs/plugin/localizedFormat'
dayjs.extend(localizedFormat)

const date = dayjs('2024-01-15')

// US English (default)
date.locale('en').format('L') // '1/15/2024'

// German
date.locale('de').format('L') // '15.01.2024'

// Japanese
date.locale('ja').format('L') // '2024/1/15'

// Spanish
date.locale('es').format('L') // '15/1/2024'

// Long formats
date.locale('en').format('LL') // 'January 15, 2024'
date.locale('de').format('LL') // '15. Januar 2024'
date.locale('ja').format('LL') // '2024年1月15日'
```

### Weekday Names

```javascript
import 'dayjs/locale/es'
import 'dayjs/locale/fr'

const date = dayjs('2024-01-15') // Monday

// Full weekday names
date.locale('en').format('dddd') // 'Monday'
date.locale('es').format('dddd') // 'lunes'
date.locale('fr').format('dddd') // 'lundi'

// Short weekday names
date.locale('en').format('ddd') // 'Mon'
date.locale('es').format('ddd') // 'lun'
date.locale('fr').format('ddd') // 'lun'

// Min weekday names
date.locale('en').format('dd') // 'Mo'
date.locale('de').format('dd') // 'Mo'
```

### Month Names

```javascript
import 'dayjs/locale/es'
import 'dayjs/locale/de'

const date = dayjs('2024-01-15')

// Full month names
date.locale('en').format('MMMM') // 'January'
date.locale('es').format('MMMM') // 'enero'
date.locale('de').format('MMMM') // 'Januar'

// Short month names
date.locale('en').format('MMM') // 'Jan'
date.locale('es').format('MMM') // 'ene'
date.locale('de').format('MMM') // 'Jan'
```

## Relative Time by Locale

Requires `relativeTime` plugin:

```javascript
import dayjs from 'dayjs'
import relativeTime from 'dayjs/plugin/relativeTime'
import 'dayjs/locale/es'
import 'dayjs/locale/de'
dayjs.extend(relativeTime)

const pastDate = dayjs().subtract(2, 'day')

// English
pastDate.locale('en').fromNow() // '2 days ago'

// Spanish
pastDate.locale('es').fromNow() // 'hace 2 días'

// German
pastDate.locale('de').fromNow() // 'vor 2 Tagen'

// Future dates
const futureDate = dayjs().add(1, 'month')
futureDate.locale('en').fromNow() // 'in a month'
futureDate.locale('es').fromNow() // 'en un mes'
```

## Week Start Configuration

Different locales start weeks on different days:

```javascript
import dayjs from 'dayjs'
import 'dayjs/locale/de' // Germany starts week on Monday
import 'dayjs/locale/en' // US starts week on Sunday

// German locale (Monday = first day)
dayjs.locale('de')
dayjs('2024-01-15').startOf('week') // 2024-01-15 (Monday)

// English locale (Sunday = first day)  
dayjs.locale('en')
dayjs('2024-01-15').startOf('week') // 2024-01-14 (Sunday)

// Custom week start
dayjs.locale('custom', {
  weekStart: 1 // Monday
})
```

## Bundle Size Optimization

Only import locales you need to keep bundle size small:

**❌ Don't load all locales:**
```javascript
import dayjs from 'dayjs'
// This loads ALL locales (~50KB)
```

**✅ Do load on-demand:**
```javascript
import dayjs from 'dayjs'
import 'dayjs/locale/es' // Only Spanish (~2KB)
import 'dayjs/locale/fr' // Only French (~2KB)
```

**Dynamic loading (advanced):**
```javascript
import dayjs from 'dayjs'

async function loadLocale(localeCode) {
  const locale = await import(`dayjs/locale/${localeCode}`)
  dayjs.locale(localeCode)
  return dayjs
}

const esDayjs = await loadLocale('es')
esDayjs().format('LLLL')
```

## Common Locale Issues

### Locale Not Working

**Problem:** Format tokens still show English names.

**Solution:** Ensure locale is imported before use:

```javascript
// ❌ Wrong order
dayjs.locale('es')
import 'dayjs/locale/es' // Too late!

// ✅ Correct order
import 'dayjs/locale/es'
dayjs.locale('es')
```

### Locale Code Mismatch

**Problem:** Using wrong locale code.

**Solution:** Check exact locale code in Day.js:

```javascript
// ❌ Wrong codes
dayjs.locale('spanish')    // Doesn't exist
dayjs.locale('chinese')    // Doesn't exist

// ✅ Correct codes
dayjs.locale('es')         // Spanish
dayjs.locale('zh-cn')      // Chinese Simplified
dayjs.locale('zh-tw')      // Chinese Traditional
```

### Plugin Not Loaded

**Problem:** Relative time shows numbers instead of text.

**Solution:** Load relativeTime plugin:

```javascript
import relativeTime from 'dayjs/plugin/relativeTime'
dayjs.extend(relativeTime)

// Now works with locales
dayjs().subtract(1, 'day').locale('es').fromNow() // 'hace un día'
```

## Contributing Locales

To add a new locale to Day.js:

1. Create locale file following existing patterns
2. Include all required fields (weekdays, months, etc.)
3. Submit pull request to [Day.js GitHub](https://github.com/iamkun/dayjs)

Locale template:

```javascript
export default {
  weekdays: 'Sunday_Monday_Tuesday_Wednesday_Thursday_Friday_Saturday'.split('_'),
  weekdaysShort: 'Sun_Mon_Tue_Wed_Thu_Fri_Sat'.split('_'),
  weekdaysMin: 'Su_Mo_Tu_We_Th_Fr_Sa'.split('_'),
  months: 'January_February_March_April_May_June_July_August_September_October_November_December'.split('_'),
  monthsShort: 'Jan_Feb_Mar_Apr_May_Jun_Jul_Aug_Sep_Oct_Nov_Dec'.split('_'),
  weekStart: 0,
  relativeTime: {
    future: 'in %s',
    past: '%s ago',
    s: 'a few seconds',
    m: 'a minute',
    mm: '%d minutes',
    h: 'an hour',
    hh: '%d hours',
    d: 'a day',
    dd: '%d days',
    M: 'a month',
    MM: '%d months',
    y: 'a year',
    yy: '%d years'
  },
  formats: {
    LT: 'h:mm A',
    LTS: 'h:mm:ss A',
    L: 'MM/DD/YYYY',
    LL: 'MMMM D, YYYY',
    LLL: 'MMMM D, YYYY h:mm A',
    LLLL: 'dddd, MMMM D, YYYY h:mm A'
  },
  invalidDate: 'Invalid Date'
}
```
