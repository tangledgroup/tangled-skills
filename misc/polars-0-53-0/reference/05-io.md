## CSV

### Read

```python
# Basic read
df = pl.read_csv("data.csv")

# With options
df = pl.read_csv(
    "data.csv",
    separator=";",
    has_header=True,
    try_parse_dates=True,
    schema={"price": pl.Float64},
    columns=["name", "price"],      # only read specific columns
    n_rows=1000,                     # limit rows read
)

# Lazy scan (preferred for large files)
lf = pl.scan_csv("data.csv")
```

### Write

```python
df.write_csv(
    "output.csv",
    include_header=True,
    separator=",",
)
```

## Parquet

Parquet is the recommended format — columnar, compressed, schema-preserving.

### Read

```python
# Basic read
df = pl.read_parquet("data.parquet")

# Lazy scan with glob
lf = pl.scan_parquet("data/2024-*/part-*.parquet")

# Read specific columns
df = pl.read_parquet("data.parquet", columns=["id", "name"])

# With row index
df = pl.read_parquet("data.parquet", row_index_name="row_num")
```

### Write

```python
df.write_parquet(
    "output.parquet",
    compression="zstd",          # snappy, gzip, zstd, lz4
    statistics=True,
    row_group_size=500_000,
)
```

## JSON (NDJSON)

Polars reads newline-delimited JSON (one JSON object per line):

```python
# Read
df = pl.read_ndjson("data.jsonl")
lf = pl.scan_ndjson("data.jsonl")

# Write
df.write_ndjson("output.jsonl")
```

Standard JSON arrays require conversion via PyArrow:

```python
import json
with open("data.json") as f:
    data = json.load(f)
df = pl.DataFrame(data)
```

## Excel

Requires `polars[excel]` feature. Multiple engines available:

```python
# Read (calamine engine — fastest, read-only)
df = pl.read_excel("data.xlsx", engine="calamine", sheet_name="Sheet1")

# Read with openpyxl
df = pl.read_excel("data.xlsx", engine="openpyxl", sheet_id=0)

# Write (xlsxwriter engine)
df.write_excel(
    "output.xlsx",
    worksheet="Results",
    engine="xlsxwriter",
)
```

## Databases

Requires `polars[database]` feature for full support.

### Read from databases

```python
# Using ADBC (requires 'adbc' feature)
df = pl.read_database_uri(
    "SELECT * FROM orders WHERE status = 'active'",
    "postgresql://user:pass@host/db",
)

# Lazy scan
lf = pl.scan_database_uri(
    "SELECT * FROM orders",
    "postgresql://user:pass@host/db",
)

# Using ConnectorX (requires 'connectorx' feature)
df = pl.read_database_uri(
    "SELECT * FROM orders",
    "mysql://user:pass@host/db",
    engine="connectorx",
)
```

### Write to databases

```python
# Using SQLAlchemy (requires 'sqlalchemy' feature)
from sqlalchemy import create_engine

engine = create_engine("postgresql://user:pass@host/db")
df.write_database("orders_table", engine, if_exists="append")

# Lazy sink
lf.sink_database(
    table="orders_table",
    connection=engine,
)
```

## Cloud Storage

Requires `polars[fsspec]` feature. Uses standard URI schemes:

```python
# S3
df = pl.read_parquet("s3://my-bucket/data/*.parquet")
lf = pl.scan_parquet("s3://my-bucket/data/part-*.parquet")

# Google Cloud Storage
df = pl.read_csv("gs://my-bucket/data.csv")

# Azure Blob
df = pl.read_parquet("az://container/path/file.parquet")
```

Configure credentials via environment variables or fsspec:

```python
import fsspec

filesystem = fsspec.filesystem("s3", key="AKIA...", secret="secret...")
df = pl.read_parquet("s3://bucket/data.parquet", storage_options={"anon": False})
```

## Arrow IPC

Zero-copy interchange with PyArrow:

```python
# Write IPC
df.write_ipc("data.arrow")
df.write_ipc("data.feather", compression="lz4")

# Read IPC
df = pl.read_ipc("data.arrow")
lf = pl.scan_ipc("data.arrow")
```

## Multiple File Scanning

Scan multiple files as a single LazyFrame with glob patterns:

```python
# All parquet files in directory
lf = pl.scan_parquet("data/*.parquet")

# Nested glob
lf = pl.scan_parquet("data/2024-*/part-*.parquet")

# Concatenate multiple sources
lf1 = pl.scan_csv("source1.csv")
lf2 = pl.scan_csv("source2.csv")
combined = pl.concat([lf1, lf2])
```

For diagonal concatenation (different schemas), use `how="diagonal"`:

```python
combined = pl.concat([df1, df2], how="diagonal")
```
