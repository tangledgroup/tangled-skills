## Universal Properties as a Method

A **universal property** characterizes an object by its relationships to other objects, not internal structure. This is the core method of category theory: define objects "up to unique isomorphism" via how morphisms factor through them.

| Perspective | Description |
|-------------|-------------|
| **Math** | Object *U* with morphisms such that any compatible object factors uniquely through *U* |
| **Code** | Interface contract: "given any implementation satisfying these constraints, there exists a unique mapping to mine." API design by specification, not implementation |

The pattern: given a diagram, find *U* with morphisms to/from it such that any other compatible object factors uniquely through *U*.

## Products and Coproducts

**Product** *A × B*: object with projections *π*1 : *A × B* → *A*, *π*2 : *A × B* → *B*. For any *X* with *f* : *X* → *A*, *g* : *X* → *B*, unique *⟨f,g⟩* : *X* → *A × B* factors through projections.

**Coproduct** *A + B* (dual): injections *ι*1 : *A* → *A + B*, *ι*2 : *B* → *A + B*. For any *X* with *f* : *A* → *X*, *g* : *B* → *X*, unique *[f,g]* : *A + B* → *X*.

| Perspective | Description |
|-------------|-------------|
| **Math** | Product = limit of discrete diagram; coproduct = colimit of discrete diagram |
| **Code** | Product types: `(a, b)` tuples. Coproduct types: `Either a b` tagged unions. ADTs are built from products + coproducts |

**In Set:** product = cartesian product, coproduct = disjoint union.

## Pullbacks and Pushouts

**Pullback** of *f* : *A* → *C*, *g* : *B* → *C*: object *P* with *p*1 : *P* → *A*, *p*2 : *P* → *B* such that *f* ∘ *p*1 = *g* ∘ *p*2, universal among such squares.

**In Set:** *P* = {(*a*, *b*) ∈ *A × B* | *f(a)* = *g(b)*}. The fibered product.

**Pushout** (dual): glues *A* and *B* along common image of *C*. In **Set**, quotient of disjoint union identifying *f(c)* with *g(c)*.

## Equalizers and Coequalizers

**Equalizer** of *f*, *g* : *A* → *B*: object *E* with *e* : *E* → *A* such that *f* ∘ *e* = *g* ∘ *e*, universal.

**In Set:** *E* = {*a* ∈ *A* | *f(a)* = *g(a)*}, inclusion map.

**Kernel** (additive categories): equalizer of *f* and the zero morphism.

**Coequalizer** (dual): *Q* with *q* : *B* → *Q* such that *q* ∘ *f* = *q* ∘ *g*. In **Set**, quotient by equivalence relation generated by *f(a)* ~ *g(a)*.

## General Limits and Colimits

A **limit** of diagram *D* : **J** → **C** is a universal cone from object *L* to the diagram. A **colimit** is the dual — universal cocone.

| Construction | Limit of | Colimit of |
|-------------|----------|------------|
| Terminal object | empty diagram | — |
| Initial object | — | empty diagram |
| Product | discrete diagram | — |
| Coproduct | — | discrete diagram |
| Pullback | cospan | — |
| Pushout | — | span |
| Inverse limit | directed (backward) | — |
| Direct limit | — | directed |

A category is **complete** if all small limits exist, **cocomplete** if all small colimits exist.

## Adjunctions

An **adjunction** *F* ⊣ *G* is a pair of functors *F* : **C** → **D**, *G* : **D** → **C** with natural bijection:

Hom_D(*F(X)*, *Y*) ≅ Hom_C(*X*, *G(Y)*)

*F* is the **left adjoint**, *G* is the **right adjoint**.

| Perspective | Description |
|-------------|-------------|
| **Math** | Optimal approximation of an inverse: *F* is "as close to inverse of G as possible" |
| **Code** | Free/forgetful patterns: `free(list) ⊣ forget(to_elements)`. Parsing ⊣ printing. Serialization ⊣ deserialization |

**Unit** *η* : Id_C ⇒ *GF*, **counit** *ε* : *FG* ⇒ Id_D. Triangle identities: *εF* ∘ *Fη* = Id_F, *Gε* ∘ *ηG* = Id_G.

**Key examples:**
- Free monoid ⊣ Forgetful (Mon → Set). Free monoid on *S* = finite lists of *S*.
- Tensor-Hom: Hom(*V ⊗ W*, *U*) ≅ Hom(*V*, Hom(*W*, *U*)) in **Vect**.
- Product/Exponential: Hom(*A × B*, *C*) ≅ Hom(*A*, *C^B*) in cartesian closed categories.

Adjunctions are the abstract view of universal properties. Every universal property gives an adjunction.

## The Yoneda Lemma

The **Yoneda embedding** maps *X* to Hom(*X*, –) : **C** → **Set**. The **Yoneda lemma**:

Nat(Hom(*X*, –), *F*) ≅ *F(X)*

Natural transformations from representable Hom(*X*, –) to *F* correspond one-to-one with elements of *F(X)*.

**Corollary:** Nat(Hom(*X*, –), Hom(*Y*, –)) ≅ Hom(*Y*, *X*). Two objects are isomorphic iff their representable functors are naturally isomorphic.

| Perspective | Description |
|-------------|-------------|
| **Math** | An object is completely determined by its relationships to all other objects |
| **Code** | A data type is fully characterized by how it can be constructed from and mapped to other types. The "API surface" determines the type |

## Representable Functors

A functor *F* : **C**op → **Set** is **representable** if naturally isomorphic to Hom(–, *X*) for some *X* (the **representing object**).

Representable functors preserve all limits that exist in **C** (as contravariant functors, they turn colimits into limits).

**Example:** In **Vect**, the dual space functor *V* ↦ Hom(*V*, *k*) is representable by the base field *k*.
