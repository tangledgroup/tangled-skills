# I/O and Memory Mapping

## NumPy Binary Formats

### .npy — Single Array

The `.npy` format stores a single array with header metadata:

```python
import numpy as np

a = np.array([[1, 2, 3], [4, 5, 6]])

# Save
np.save('data.npy', a)

# Load
b = np.load('data.npy')

# With allow_pickle control (security consideration)
c = np.load('data.npy', allow_pickle=False)
```

### .npz — Compressed Archive

Multiple arrays in a single compressed file:

```python
# Save multiple arrays
np.savez('bundle.npz', x=a, y=np.arange(10), z=np.random.rand(5, 5))

# Save with compression (smaller files)
np.savez_compressed('bundle_comp.npz', x=a, y=np.arange(10))

# Load — returns NpzFile (dict-like, lazy loading)
data = np.load('bundle.npz')
data['x']     # array x
data['y']     # array y
list(data.keys())  # ['x', 'y', 'z']

# Context manager for automatic cleanup
with np.load('bundle.npz') as data:
    print(data['x'])
```

## Text File I/O

### np.loadtxt

Simple text files with uniform data:

```python
# Load from CSV
data = np.loadtxt('data.csv', delimiter=',', dtype=float)

# Skip header rows
data = np.loadtxt('data.csv', delimiter=',', skiprows=1)

# Select specific columns
data = np.loadtxt('data.csv', delimiter=',', usecols=(0, 2, 4))

# With comments
data = np.loadtxt('data.txt', delimiter='\t', comments='#')
```

### np.savetxt

```python
a = np.random.rand(5, 3)

# Basic save
np.savetxt('output.csv', a, delimiter=',')

# With header and formatting
np.savetxt('output.csv', a, delimiter=',',
           header='col1,col2,col3',
           comments='# ',
           fmt='%.6f')
```

### np.genfromtxt

Handles missing values and mixed types:

```python
# Auto-detect dtype, handle missing values
data = np.genfromtxt('data.csv', delimiter=',', missing_values='?', filling_value=np.nan)

# With names from header
data = np.genfromtxt('data.csv', delimiter=',', names=True, dtype=None)

# Specific column types
data = np.genfromtxt('data.csv', delimiter=',',
                     dtype=[('name', 'U20'), ('value', 'f8')])
```

## Raw Binary I/O

```python
# From raw binary file
a = np.fromfile('data.bin', dtype=np.float32)

# Write to raw binary
a.tofile('output.bin')

# With offset and count
partial = np.fromfile('data.bin', dtype=np.float32, count=100, offset=4096)
```

## Memory-Mapped Arrays

Memory mapping allows working with arrays larger than RAM by mapping them to disk:

```python
# Create a memory-mapped file
m = np.memmap('large_array.dat', dtype=np.float64, mode='w+', shape=(1000000, 1000))

# Use like a regular array (data paged in/out automatically)
m[0:100, :] = np.random.rand(100, 1000)
m.flush()  # write changes to disk

# Open existing memmap for reading
m = np.memmap('large_array.dat', dtype=np.float64, mode='r', shape=(1000000, 1000))

# Mode options: 'r' (read-only), 'r+' (read/write), 'w+' (create/open), 'c' (copy-on-write)
```

### memmap with np.lib.format

```python
from numpy.lib.format import open_memmap

# Create
m = open_memmap('data.dat', dtype='f8', mode='w+', shape=(1000, 1000))

# Open existing
m = open_memmap('data.dat', mode='r+')
```

## String Formatting

### Array to String

```python
a = np.array([1.23456789, 2.34567890, 3.45678901])

np.array_str(a)           # default string representation
np.array_repr(a)          # repr-style (includes dtype info)
np.array2string(a, precision=4, separator=', ', prefix='[', suffix=']')
```

### Float Formatting Functions

```python
np.format_float_positional(3.14159, precision=2)    # '3.14'
np.format_float_scientific(0.00123, precision=3)     # '1.230e-03'
```

## Print Options

```python
# Set global print options
np.set_printoptions(precision=4, threshold=100, linewidth=80, suppress=True)

# Get current options
opts = np.get_printoptions()

# Context manager for temporary settings
with np.printoptions(precision=2, threshold=5):
    print(large_array)  # uses temporary settings

# Custom formatters
np.set_printoptions(formatter={
    'float_kind': '{:0.2f}'.format,
    'int_kind': '{:d}'.format,
})
```

## Base-N Representations

```python
np.binary_repr(13)           # '1101'
np.binary_repr(-13, width=8) # '-01101' (padded)
np.base_repr(100, base=16)   # '64' (hexadecimal)
np.base_repr(100, base=2)    # '1100100'
```

## DataSource Utility

Generic data source supporting file, HTTP, and FTP:

```python
from numpy.lib.npyio import DataSource

ds = DataSource('https://example.com/data.csv')
with ds.getfile('data.csv') as f:
    data = np.loadtxt(f, delimiter=',')
```

## I/O Best Practices

1. Use `.npy`/`.npz` for NumPy-native storage (fast, preserves dtype)
2. Use `memmap` for arrays larger than available RAM
3. Use `genfromtxt` when data has missing values or mixed types
4. Set `allow_pickle=False` on `np.load` unless you need object arrays
5. Use `savez_compressed` for storage efficiency with repeated patterns
6. For very large datasets, consider HDF5 (via h5py) or Zarr instead of NumPy native formats
